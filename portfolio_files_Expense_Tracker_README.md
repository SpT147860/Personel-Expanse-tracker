# Expense Tracker: Python + CSV/Pandas

A personal finance tool that tracks, categorizes, and visualizes monthly spending. This project demonstrates data persistence using CSV files and data visualization with Matplotlib.

## Features
- **Data Entry**: Input date, category, and amount for each expense.
- **Data Persistence**: Automatically saves data to a `expenses.csv` file.
- **Visualization**: Generates a pie chart (`expense_report.png`) showing spending distribution.

## Requirements
- Python 3.x
- Pandas
- Matplotlib

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Expense-Tracker.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the script and follow the on-screen prompts:
```bash
python expense_tracker.py
```
